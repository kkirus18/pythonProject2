# 📌 Генератор тестов

> Программа для генерации тестовых заданий с поддержкой двух режимов: «Обучение» (с подсказками) и «Экзамен» (без подсказок). База вопросов содержит 20+ вопросов по языкознанию и лингвистике. Каждый пользователь получает уникальный вариант теста со случайным порядком вопросов и вариантов ответов.

## 🚀 Как запустить проект

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/kkirus18/pythonProject2.git
   cd pythonProject2
2. Создайте виртуальное окружение:

bash
python -m venv .venv
3. Активируйте окружение:

Windows: .venv\Scripts\activate

Linux/macOS: source .venv/bin/activate

4. Запустите программу:

bash
python main.py

🎮 Функциональность
База вопросов: 20+ вопросов по лингвистике и языкам

Режим «Обучение»: подсказки после неправильных ответов

Режим «Экзамен»: строгий режим без подсказок

Уникальный вариант: каждый тест имеет ID (дата+время+случайное число)

Перемешивание: вопросы и варианты ответов в случайном порядке

Разбор ошибок: после завершения показываются правильные ответы
