# 📌 Генератор тестов

> Программа для генерации тестовых заданий с поддержкой двух режимов: «Обучение» (с подсказками) и «Экзамен» (без подсказок). База вопросов содержит 20+ вопросов по языкознанию и лингвистике. Каждый пользователь получает уникальный вариант теста со случайным порядком вопросов и вариантов ответов.
